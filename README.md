# 취뽀 프로젝트
Python + Django + Selenium + Mysql
<br><br>


## 🖥️ 프로젝트 소개
사람인, 잡코리아, 잡플래닛, 인크루트 등 N개의 채용 사이트의 구직 정보를 통합해서 제공하고, 
하나의 대시보드에서 취업 준비의 시작부터 끝까지 마무리할 수 있는 웹서비스를 제공하는 취뽀 사이트 입니다. 
<br><br>

### 🕰️ 개발 기간
* 23.06.26 ~ 23.07.07 (2주)

### 🧑‍🤝‍🧑 멤버구성
 - 개발팀 2명, 분석팀 1명

### ⚙️ 개발 환경
- `Python 3.10.9`
- **IDE** : PyCharm (Local)
- **Framework** : Django (4.1)
- **Database** : MySQL (5.7.21)

<br>
